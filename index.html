Sign Aloud Gloves

A Gesture to Speech System that converts hand movements into voice output to help mute and speech-impaired people communicate effectively.

School: Nabagram Vidyapith

Project By: Aritra Nath & Sohom Tambuly

About the Project

Sign Aloud Gloves is an assistive technology project built using Arduino. It detects hand gestures using a gyroscope sensor and touch sensors, then converts those gestures into pre‑recorded voice messages using a DF Player Mini.

Objectives

  • Help mute and speech‑impaired people express their thoughts
  • Convert hand gestures into audible speech
  • Create a low‑cost and portable assistive device

Hardware Components

Arduino Uno
MPU‑6050 Gyroscope & Accelerometer
Touch Sensors (2x)
DF Player Mini
Speaker
SD Card with Audio Files

Project Code

#include <Wire.h>
#include <Adafruit_MPU6050.h>
#include <Adafruit_Sensor.h>
#include <LiquidCrystal_I2C.h>
#include <SoftwareSerial.h>
#include <DFRobotDFPlayerMini.h>

// Objects
Adafruit_MPU6050 mpu;
LiquidCrystal_I2C lcd(0x27, 16, 2);

// DFPlayer
SoftwareSerial mySerial(10, 11);
DFRobotDFPlayerMini dfplayer;

// Touch Sensors
const int s1 = 3;
const int s2 = 4;
const int s3 = 5;
const int s4 = 6;

int lastTrack = 0;

void playVoice(int track) {
  if (track != lastTrack) {
    dfplayer.play(track);
    lastTrack = track;
  }
}

void setup() {
  Serial.begin(115200);
  mySerial.begin(9600);

  pinMode(s1, INPUT);
  pinMode(s2, INPUT);
  pinMode(s3, INPUT);
  pinMode(s4, INPUT);

  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0);
  lcd.print("Smart Vocalizer");

  if (!mpu.begin()) {
    lcd.setCursor(0, 1);
    lcd.print("MPU6050 Error");
    while (1);
  }

  if (!dfplayer.begin(mySerial)) {
    lcd.clear();
    lcd.print("DFPlayer Error");
    while (1);
  }

  dfplayer.volume(30);
  delay(2000);
  lcd.clear();
}

void loop() {
  sensors_event_t accel, gyro, temp;
  mpu.getEvent(&accel, &gyro, &temp);

  float roll = atan2(accel.acceleration.y, accel.acceleration.z) * 180 / PI;
  float pitch = atan2(-accel.acceleration.x,
    sqrt(accel.acceleration.y * accel.acceleration.y +
         accel.acceleration.z * accel.acceleration.z)) * 180 / PI;

  int s1_out = digitalRead(s1);
  int s2_out = digitalRead(s2);

  lcd.clear();
  lcd.setCursor(0, 1);

  if (pitch > 30) {
    lcd.print("I'm in pain");
    playVoice(1);
  } else if (pitch < -30) {
    lcd.print("I need help");
    playVoice(2);
  } else if (roll > 30) {
    lcd.print("No");
    playVoice(8);
  } else if (roll < -30) {
    lcd.print("Please wait");
    playVoice(4);
  } else {
    if (s1_out == HIGH && s2_out == HIGH) {
      lcd.print("Call my family");
      playVoice(7);
    } else if (s1_out == HIGH) {
      lcd.print("Please stop");
      playVoice(5);
    } else if (s2_out == HIGH) {
      lcd.print("Yes");
      playVoice(6);
    } else {
      lcd.print("hello");
    }
  }
  delay(500);
}

Working Principle

The MPU‑6050 sensor detects the orientation and movement of the hand. Touch sensors are used to trigger or confirm gestures. Based on the detected gesture, Arduino sends a command to the DF Player Mini, which plays the corresponding audio message through a speaker.

Applications

  • Communication aid for mute or speech‑impaired individuals
  • Educational demonstration of gesture recognition
  • Human‑Computer Interaction projects

Future Improvements
